# Credit Card Financial Dashboard

## Overview

This project focuses on analyzing credit card customer and transaction data using Power BI. An interactive dashboard was developed to monitor revenue, transaction volume, customer demographics, spending patterns, and business performance metrics. The dashboard provides valuable insights that support data-driven decision-making and business growth.

---

## Features

* Interactive Power BI Dashboard
* Revenue Analysis
* Customer Segmentation
* Transaction Analysis
* KPI Monitoring
* Weekly Performance Tracking
* Dynamic Filters and Slicers
* Data Visualization and Reporting

---
